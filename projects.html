Featured project

PhisDefense SOC and Email Security Lab

PhisDefense is a defensive cybersecurity case study focused on email security, Blue Team monitoring and SOC-style evidence handling. The project was built around a real mail security environment and validates security controls through technical evidence, controlled tests and a public interactive dashboard.

The lab includes Postfix, Dovecot, OpenDKIM, OpenDMARC, OpenARC, SPF, DKIM, DMARC p=reject, DNSSEC, MTA-STS, TLS-RPT, STARTTLS and IMAPS TLS. It also includes Python/Dash dashboarding to visualize processed SOC events and security test results.
