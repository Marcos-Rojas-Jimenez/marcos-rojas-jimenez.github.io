About Me

Defensive cybersecurity profile with a technical systems background.

I am a junior cybersecurity professional focused on defensive security, SOC monitoring, Blue Team operations and email security. Featured case study

PhisDefense SOC and Email Security Lab

View full case study

Defensive email security and SOC monitoring project built around a real mail security environment. It validates SPF, DKIM, a progressive DMARC policy ending in p=reject, DNSSEC, MTA-STS, TLS-RPT, SMTP security controls and SOC-style evidence handling through a public dashboard.

270 security tests
190 SOC events
p=reject final DMARC policy
s2026 DKIM rotation selector
Email Security Blue Team SOC Linux Postfix Dovecot OpenDKIM OpenDMARC OpenARC Python Dash
Project objective

Build a defensive email security lab where controls were not only configured, but also tested, documented and validated with technical evidence.

Technical environment

Real Ubuntu VPS environment using Postfix, Dovecot, OpenDKIM, OpenDMARC, OpenARC, DNS records, TLS validation, syslog evidence and Python-based processing.

Validated controls

SPF fail, DKIM valid/broken/absent, DKIM rotation to s2026, progressive DMARC enforcement to p=reject, DNSSEC, MTA-STS, TLS-RPT, STARTTLS SMTP, IMAPS TLS and spoofing rejection.

SOC evidence and dashboard

Server events and controlled tests were processed into a sanitized SOC dataset and visualized through an interactive Python/Dash dashboard.

Public version and privacy

The public version removes private keys, credentials, full raw logs, internal paths, sensitive IPs and complete server backups.
